# DesignDead — VS Code Extension

Visual editor for `.dd` project files created by [DesignDead](https://github.com/withso/design-dead).

## Features

- **Visual Canvas** — Preview all design variants with live HTML/CSS rendering
- **Project Overview** — See project metadata, breakpoints, and workspace config
- **Variant Cards** — Each variant rendered in an isolated iframe preview
- **Feedback Panel** — View and manage feedback items per variant
- **Open as Text** — Fallback to raw JSON editing anytime

## Usage

1. Install the extension
2. Open any `.dd` file in VS Code
3. The visual editor opens automatically
4. Use "Open as Text" (toolbar button) to view raw JSON

## Compatibility

Works with VS Code, Cursor, and Windsurf.
